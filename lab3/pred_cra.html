RMSE¶

Duas funções úteis para verificarmos a melhoria dos modelos após mudar os parâmetros nas contruções.

In [354]:
def rmse_cv(model):
    rmse= np.sqrt(-cross_val_score(model, X_train, y, scoring="neg_mean_squared_error", cv = 5))
    return(rmse)

def rmse(y_actual,y_predicted):
    return sqrt(mean_squared_error(y_actual, y_predicted))

Modelos tunados¶

Nesta sessão os modelos foram analisados para obter os melhores alphas, utilizando regularização, para Lasso e Ridge, e no caso do K-nearest-Neighbors o melhor N.

Além dos parâmetros que foram modificados, duas variáveis foram criadas, sendo elas: p1-lp1 e p2-lp2. O principal motivo da criação dessas variáveis é que as notas dessas disciplinas são dependentes e talvez se analisadas em separado não tenham tanto peso para predição do CRA como quando analisadas em conjunto.

In [361]:
